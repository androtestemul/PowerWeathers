# PowerWeathers

Написать погодное Android приложение на Kotlin.

Требования:

Весь интерфейс написать программно без использования XML.
Отобразить один экран с погодной информацией: текущая, почасовая (для текущего дня) и прогноз погоды (3 дня) для Москвы
Обработать показ индикатора загрузки, стандартного диалога при возникновении ошибки с возможностью повторить запрос.

UI:

По дизайну никаких ограничений нет, можно ориентироваться на любое стандартное погодное приложение или примеры из интернета, и сделать его в упрощенном стиле.
